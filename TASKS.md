# Task List — Sardegna Eventi

Scrivi qui i task che vuoi vengano eseguiti. Quando mi dici "fai i task" o simile, leggo questo file e li eseguo nell'ordine indicato.

## Come scrivere un task

Ogni task deve avere:
- **ID**: numero progressivo (es. T001)
- **Stato**: `TODO` | `IN_PROGRESS` | `DONE` | `BLOCKED`
- **Descrizione**: cosa fare, in italiano o inglese
- **Priorità**: `ALTA` | `MEDIA` | `BASSA` (opzionale)
- **Note**: dettagli extra, vincoli, o "vedi anche X" (opzionale)
